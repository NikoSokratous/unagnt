# v0.5 Quick Reference Card

## 🎯 Policy Engine (Phase 1)

```bash
# List policies
unagnt policy list

# Test policy
unagnt policy test security_test.yaml

# Simulate changes
unagnt policy simulate security.yaml delete_file

# Query audit logs
unagnt policy audit query --decision deny --days 7

# Export audit logs
unagnt policy audit export --format csv --output audit.csv
```

## 🔁 Deterministic Replay (Phase 2)

```bash
# List snapshots
unagnt replay list

# Exact replay (fast, deterministic)
unagnt replay run snap-123 --mode exact

# Live replay (test changes)
unagnt replay run snap-123 --mode live

# Debug with breakpoints
unagnt replay debug snap-123

# Validate integrity
unagnt replay validate snap-123

# Compare snapshots
unagnt replay diff snap-old snap-new
```

## ⚖️ Risk Scoring (Phase 3)

```bash
# Assess action risk
unagnt risk assess --tool delete_file --env production

# View statistics
unagnt risk stats --days 7

# Top risky actions
unagnt risk top --limit 10 --min-score 0.7

# Trend analysis
unagnt risk trend --days 30 --interval day

# Generate compliance report
unagnt risk report generate --type daily

# View report
unagnt risk report view report-001

# Export report
unagnt risk report export report-001 --format json
```

```bash

unagnt policy activate my-policy 1.0.0
```

### Debug Failed Run
